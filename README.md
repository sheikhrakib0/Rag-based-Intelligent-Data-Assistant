```bash
## ğŸ“¡ API Endpoints
1. Upload Document

POST /upload

Description: Uploads a file (PDF, CSV, or TXT), extracts text, generates embeddings, and stores them in FAISS.

Request (form-data):

Key	Type	Description
file	File	The document to upload

Example (using cURL):

curl -X 'POST' \
  'http://127.0.0.1:8000/upload' \
  -F 'file=@example.pdf'


Example Response:

{
  "message": "âœ… File processed successfully!",
  "file_id": "e97a1fce-5a61-4e8b-9a23-f7c95acb6b17",
  "filename": "example.pdf",
  "stored_in": "data/vector_db"
}

ğŸ”µ 2. Ask Question

POST /ask

Description: Retrieves relevant chunks from FAISS and generates a contextual answer using Google Gemini. Use db_path: data/vector_db

Request (JSON):

{
  "query": "Summarize this document.",
  "db_path": "data/vector_db"
}


Example Response:

{
  "query": "Summarize this document.",
  "response": "This document discusses the role of AI in business innovation...",
  "retrieved_chunks": [
    "Artificial intelligence is transforming industries by automating tasks...",
    "Machine learning enables predictive analytics for better decision-making..."
  ]
}

```
