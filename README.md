# 📘 RAG Based Intelligent Data Assistant with FastAPI & Google Gemini

A **Retrieval-Augmented Generation (RAG)** web API built using **FastAPI**, **LangChain**, and **Google Gemini**.  
This project allows you to **upload documents**, **store their embeddings**, and **ask natural language questions** about the uploaded content. 

Check out live API: [RAG Based Intelligent Data Assistant](https://rag-based-intelligent-data-assistant-uno2.onrender.com/docs#/)

---

## 🚀 Features

✅ Upload and process PDF, CSV, or TXT files  
✅ Extract text content from documents

✅ Used Langchain Framework for this project  
✅ Generate vector embeddings using **Gemini Embedding Model**  
✅ Store and retrieve vectors using **FAISS**  
✅ Query with context-aware answers powered by **Google Gemini** via **LangChain**  
✅ Modular code structure for easy extension and deployment

```bash
## 📡 API Endpoints
1. Upload Document

POST /upload

Description: Uploads a file (PDF, CSV, or TXT), extracts text, generates embeddings, and stores them in FAISS.

Request (form-data):

Key	Type	Description
file	File	The document to upload

Example (using cURL):

curl -X 'POST' \
  'http://127.0.0.1:8000/upload' \
  -F 'file=@example.pdf'


Example Response:

{
  "message": "✅ File processed successfully!",
  "file_id": "e97a1fce-5a61-4e8b-9a23-f7c95acb6b17",
  "filename": "example.pdf",
  "stored_in": "data/vector_db"
}

🔵 2. Ask Question

POST /ask

Description: Retrieves relevant chunks from FAISS and generates a contextual answer using Google Gemini. Use db_path: data/vector_db

Request (JSON):

{
  "query": "Summarize this document.",
  "db_path": "data/vector_db"
}


Example Response:

{
  "query": "Summarize this document.",
  "response": "This document discusses the role of AI in business innovation...",
  "retrieved_chunks": [
    "Artificial intelligence is transforming industries by automating tasks...",
    "Machine learning enables predictive analytics for better decision-making..."
  ]
}

```

## 🧩 Example Workflow

1️⃣ Upload your document

Go to /docs

Use /upload endpoint to upload a PDF or text file

Note down the "stored_in" path from the response (e.g., data/vector_db)

2️⃣ Ask questions

Use /ask endpoint

Provide a question and the same db_path returned earlier

Get a context-aware response generated by Google Gemini
